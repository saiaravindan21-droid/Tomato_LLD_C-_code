This folder contains the C++ code examples for Lecture 11 of the Low-Level Design Course.

Files included
main.cpp: Entry point for the application.
Cart.h, Order.h, OrderManager.h, RestaurantManager.h, User.h, MenuItem.h, Restaurant.h, 
OrderFactory.h, NowOrderFactory.h, ScheduledOrderFactory.h, PickupOrder.h, DeliveryOrder.h, 
PaymentStrategy.h, CreditCartPaymentStrategy.h, UpiPaymentStrategy.h, NotificationService.h, 
TimeUtils.h, TomatoApp.h, folderStructure.txt: Header files and utility files implementing various classes and functionalities for the lecture's topic.


These files demonstrate an order management system for a restaurant application, using object-oriented design principles and design patterns such as Factory and Strategy.
